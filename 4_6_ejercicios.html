Ejercicios¶

Realiza una aplicación en Flask con las siguientes características:

  • ruta: / => redirecciona a inicio
  • ruta: /inicio => Muestra Mensaje: Bienvenido a aplicación + Imagen Paradisiaca
  • ruta: /juguetes/post => Muestra Mensaje: Nuevos jueguetes, con petición POST
  • ruta: /privado => Muestra Mensaje: Error 401, zona protegida

Una vez hecho esto, dentro de la carpeta static/img/ añade 3 imagenes de juguetes. Dichos juguetes se deben denominar juguete1.png, juguete2.png y juguete3.png.

  • ruta: /juguetes => Muestra las imágenes de los 3 juguetes
  • ruta: /juguetes/ => Muestra la imagen del juguete escrito (Sin .png)
